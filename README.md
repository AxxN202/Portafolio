# 🎌 Interactive Manga Style Portfolio

Un portafolio interactivo de desarrollo web construido con HTML, CSS puro y Bootstrap 5, diseñado con una estética única inspirada en el Manga japonés (alto contraste en blanco y negro) combinada con efectos dinámicos de neón.

## Características Principales

* **Estética Manga Monocromática**: Uso inteligente de escalas de grises, blancos puros y fondos oscuros que evitan la fatiga visual mientras mantienen un estilo gráfico muy marcado.
* **Layout Interactivo**: Las tarjetas se presentan en un contenedor de cuadrícula flexible. Al interactuar con ellas, la tarjeta activa resalta y se agranda, mientras que el resto se encoge sutilmente para darle protagonismo.
* **Glow Neón Dinámico**: Integración de variables CSS manejadas por JavaScript para generar un color de resplandor (`--random-hue`) aleatorio cada vez que el usuario pasa el mouse por una tarjeta.
* **Diseño Totalmente Responsivo**: Un diseño fluido que se asegura de mostrar exactamente 4 tarjetas por fila en escritorio, ajustándose automáticamente para apilar las tarjetas en pantallas de dispositivos móviles.
* **Botones Funcionales**: Enlaces directos pre-configurados y estilizados para redirigir a WhatsApp, GitHub, LinkedIn y contacto por Email.

## Tecnologías Utilizadas

* **HTML5**: Estructura limpia y semántica.
* **CSS3 Avanzado**: Transiciones complejas, Variables CSS, `@keyframes`, pseudoclases modernas como `:has()` y funciones de color `oklch()`.
* **Bootstrap 5**: Utilizado para el sistema estructural (`d-flex`, `gap-3`), utilidades de espaciado, botones y márgenes.
* **FontAwesome 6**: Para toda la iconografía de la interfaz (herramientas, lenguajes y redes sociales).
* **JavaScript Vanilla**: Exclusivamente para la lógica matemática de la inyección dinámica de colores aleatorios en tiempo real.

## Uso e Instalación

Al ser un proyecto estático puro, no requiere instalación de dependencias de Node.js ni servidores de desarrollo complejos.

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com/AxxN202/tu-repositorio.git
   ```
2. Navega al directorio principal del proyecto.
3. Simplemente abre el archivo `index.html` en tu navegador web favorito (Chrome, Firefox, Edge).
   * *Recomendado: Utilizar la extensión **Live Server** de VSCode para una mejor experiencia visualizando cambios locales.*
